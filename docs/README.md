# Documentation

This directory is for the skill repository's own material, not for target Godot game projects. Keep this distinction in mind:

- `docs/` here = notes, retrospectives, and research that informed the skills in `skills/`.
- `docs/` in a Godot project = the target project's design and working files. The skills reference those paths (`docs/vision/`, `docs/working/`, `docs/design/`) by convention. Do not confuse the two.

## archive/

Finished research and retrospective notes from past skill iterations. Files are named `YYYY-MM-DD-<slug>.md` so they sort chronologically. They are non-installable reference material, not part of the skill install surface.
